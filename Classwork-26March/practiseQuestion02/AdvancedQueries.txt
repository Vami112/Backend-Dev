Find all students with GPA between 3.0 and 3.5
db.students.find({
  gpa: { $gte: 3.0, $lte: 3.5 }
})

$gte = greater than or equal to
$lte = less than or equal to

2. Find students enrolled in more than 5 courses

(Assuming courses is an array)

db.students.find({
  courses: { $size: 6 }
})

If “more than 5” strictly, use:

db.students.find({
  $expr: { $gt: [ { $size: "$courses" }, 5 ] }
})
3. Get top 10 students by GPA
db.students.find()
  .sort({ gpa: -1 })
  .limit(10)

-1 means descending order (highest first)

4. Count students by city
db.students.aggregate([
  {
    $group: {
      _id: "$city",
      totalStudents: { $sum: 1 }
    }
  }
])
